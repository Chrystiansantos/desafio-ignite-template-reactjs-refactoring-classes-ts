# Desafio refactoring classes JS => TS.

## Objetivo

Este desafio possuia como objetivo transpilar um app Front-end, escrita em React JS, para TS. abaixo segue uma imagem de exemplo da aplicação em ambiente de desenvolvimento.

## Demo

![Exemplo da aplicação](./imgReadme/GoRestaurant.png)

## Rodar na sua máquina:

Baixe o projeto, com o seguinte comando:
```bat
git clone https://github.com/Chrystiansantos/desafio-ignite-template-reactjs-refactoring-classes-ts.git
```

No terminal execute:

```bat
cd desafio-ignite-template-reactjs-refactoring-classes-ts
```

Execute yarn para instalar as dependências.

```bat
  yarn
```

A seguir execute, yarn server para iniciar a api, neste projeto estou utilizando json-server.

```bat
  yarn server
```

E por fim iremos executar yarn start para executar a aplicação.

```bat
  yarn start
```
